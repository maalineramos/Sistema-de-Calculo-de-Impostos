# Sistema de Cálculo de Impostos

Projeto Java orientado a objetos que calcula o imposto devido por contribuintes do tipo pessoa física e empresa, aplicando regras diferentes para cada classe.

## Funcionalidades
- Cadastro de múltiplos contribuintes via terminal
- Cálculo de imposto individual para pessoa física e empresa com polimorfismo
- Exibição do imposto pago por cada contribuinte
- Exibição do total geral de impostos arrecadados

## Tecnologias
- Java
- POO

## Como executar
1. Clone o repositório
2. Compile os arquivos `.java` da pasta `src`
3. Execute a classe `application.Program`
